XmlDbObject createChild(string path);

相対XPathのpathで指定される繰返し項目のレコードを一つRDBに作成し、
そのレコードに対応するXmlDbObjectを返します。

void putField(string path, string value);

XmlDbObjectに対応するレコードのXPathで指定されるカラムに値をRDBに格納します。

list getChildren(string path);

相対XPathのpathで指定された繰り返し項目に相当するレコードの
XmlDbObjectのコレクションを返します。

list getChildren(string path, string path1, string value1);

相対XPathのpathで指定された繰り返し項目に相当するレコードで、
相対XPathがpath1でそのXPathに入っているテキストがvalue1に等しい(likeを使用)レコードの
XmlDbObjectのコレクションを返します。

list getChildren(string path, string path1, string value1, string path2, string value2);

相対XPathのpathで指定された繰り返し項目に相当するレコード群で、
相対XPathがpath1でそのXPathに入っているテキストがvalue1に等しく(likeを使用)、
相対XPathがpath2でそのXPathに入っているテキストがvalue2に等しい(likeを使用)レコードに相当する
XmlDbObjectのコレクションを返します。

list getChildren(string path, list cond);

相対XPathのpathで指定された繰り返し項目に相当するレコードで、
condが(*, ...) -/path-1/-> value-1
                  ...
               -/path-n/-> value-n
で相対XPathがpath-iでそのXPathに入っているテキストがvalue-iに等しい(likeを使用)レコードに相当する
XmlDbObjectのコレクションを返します。

list getChildren(string path, list cond, string range_path, string range_type, string from, string to, string orderby_path, string orderby_type, list desc);

相対XPathのpathで指定された繰り返し項目に相当するレコードで、
condが(*, ...) -/path-1/-> value-1
                  ...
               -/path-n/-> value-n
でXPathがpath-iでそのXPathに入っているテキストがvalue-iに等しい(likeを使用)レコードで、
XPathがrange_path(型はintegerかstring)でその値がfromからtoまでのレコードを、
XPathがorderby_path(型はintegerかstring)でその値(型はintegerかstring)でソートした、
XmlDbObjectのコレクションを返します。

string getField(string path);

レコードに相当するXmlDbObjectから相対XPathで指定されるカラムに相当する値を取得します。

void removeField(string path);

レコードに相当するXmlDbObjectから相対XPathで指定されるカラムをRDBから削除します。

void delete();

レコードに相当するXmlDbObjectをRDBから削除します。
