postgresql のビルド$ cd <postgres-home>	($ cd /home/inamoto/altair/postgresql-9.3.2)$ ./configure$ gmake$ su$ gmake install$ cd /usr/local$ chown -R inamoto pgsql$ chgrp -R altair pgsql$ cd bin/pg_ctl -D data stop/usr/local/pgsql/lib/libpq.so* を/home/inamoto/altair/build にコピー。● mysql のビルド	● cmake のビルド	$ cd <cmake-home>		($ cd /home/inamoto/altair/etc/cmake-2.8.12.2)	$ があれば削除かリネーム。内容は以下だった。[mysqld]datadir=/var/lib/mysqlsocket=/var/lib/mysql/mysql.sockuser=mysql# Disabling symbolic-links is recommended to prevent assorted security scripts/mysql_install_db --user=mysql$ chown -R root .$ chown -R mysql data--------------------↓rootで行う。/etc/my.cnf があれば削除かリネーム。内容は以下だった。$ bin/mysqld_safe --user=mysql &↓任意の user で OK。$
